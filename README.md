Oasis Infobyte Internship – Task 1
📌 Description
The BMI (Body Mass Index) Calculator is a Python-based program developed as part of Task 1 of the Oasis Infobyte Internship. This project provides a simple and effective way to calculate BMI using user-provided weight and height, and classify the result into standard health categories. It is designed for beginners to practice core Python concepts.
📖 Project Overview
BMI is a commonly used health metric to assess body weight relative to height. This program asks the user to input their weight (in kilograms) and height (in meters), calculates the BMI using a standard formula, and categorizes the result as Underweight, Normal, Overweight, or Obese.
The project demonstrates how Python can be applied to solve real-life problems while strengthening programming fundamentals.
✨ Features
User Input Handling – Accepts weight and height as input
BMI Calculation – Uses the standard BMI formula
Health Classification – Categorizes BMI into standard ranges
Error Handling – Manages invalid or non-numeric input using exception handling
🛠️ Technologies & Concepts Used
Python 3.x
Built-in functions (input(), print())
Conditional statements (if-elif-else)
Exception handling (try-except)
Basic arithmetic operations
▶️ How It Works
User enters weight (kg) and height (m)
Input is validated to ensure positive numeric values
BMI is calculated using:
Copy code

BMI = weight / (height * height)
BMI is classified as:
Underweight: BMI < 18.5
Normal weight: 18.5 ≤ BMI < 24.9
Overweight: 25 ≤ BMI < 29.9
Obese: BMI ≥ 30
The BMI value and health category are displayed

The BMI Calculator successfully computes BMI and provides meaningful feedback to users. This project helped in understanding Python basics, logical conditions, error handling, and real-world problem solving. It serves as a strong beginner-level project for Python learners.
